---- Object 다형성 ----
** Object 클래스는 모든 클래스의 부모 클래스 **

** Object 클래스 다형성의 장점
 - 매개변수에 Object 타입을 사용하면 모든 타입을 전달할 수 있다.

** Object 클래스 다형성의 한계
 - Object 클래스는 최상위 부모이므로 다른 인스턴스의 메소드를 호출하려면
   항상 다운 캐스팅을 해야한다.
 - Object 클래스는 모든 객체를 대상으로 다형적 참조를 할 수 있다.

** 다형성을 제대로 활용하려면 다형적 참조 + 메소드 오버라이딩을 함께 사용해야 한다.
 - 다형적 참조는 가능하지만 메소드 오버라이딩이 안되기 때문에
   다형성을 활용하기에는 Object 클래스 만으로는 한계가 있다.
