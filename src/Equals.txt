---- 동일성과 동등성 ----
1. 동일성: == 연산자를 사용해서 두 객체의 참조(주소값)가 동일한 객체를 가리키는지 확인
 - 동일은 완전히 같음을 의미
 - 동일성은 자바 머신 기준으로 물리적으로 같은 메모리에 있는
   같은 객체 인스턴스인지 참조값을 확인하는 것
2. 동등성: equals()를 사용하여 두 객체가 논리적으로 동등한지 확인
 - 동등은 같은 가치나 수준을 의미
 - 동등성은 논리적으로 같은지를 확인
   보통 사람이 생각하는 기준에 맞추어 생각

ex) User a = new User("id-100") 참조 x001
    User b = new User("id-100") 참조 x002
  - 이 둘은 다른 메모리에 있는 다른 객체이지만, 회원 번호를 기준으로
    생각한다면 논리적으로 같은 회원이라고 볼 수 있는 것

** Object 클래스가 기본으로 제공하는 equals()는 == 을 제공하므로
   동등성 비교를 사용하고 싶다면 equals()를 오버라이딩하여 사용해야 한다.
